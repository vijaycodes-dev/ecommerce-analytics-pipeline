# 🚀 Ecommerce Analytics Pipeline

End-to-end Data Engineering project using:
- Python
- SQL
- AWS
- Airflow
- PostgreSQL
- Power BI

## 📌 Project Goal
Build a modern data pipeline for e-commerce analytics.

## 🏗️ Architecture
Raw Data → ETL → AWS S3 → PostgreSQL → Dashboard → Airflow

## 🛠️ Tech Stack
- Python
- PostgreSQL
- AWS S3
- Apache Airflow
- Power BI
